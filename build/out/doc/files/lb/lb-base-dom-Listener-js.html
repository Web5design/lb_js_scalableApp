lb.base.dom.Listener

DOM (Document Object Model) Listener Adapter Module for Base Library

Creating a Listener attaches a callback to a DOM element for a given event.  The Listener can then detach() the configured callback, avoiding memory leaks in IE.

new Listener(element, type, callback[, useCapture])

Create a new listener for a type of event on a DOM element.

Parameters

elementDOM Element, an element
typestring, the name of an event (without ‘on’) e.g.  ‘click’
callbackfunction, a function to call when the event is dispatched.
useCaptureboolean, optional, defaults to false, whether the callback is set for capture phase.  See [1] for details.

Returns

object, the new instance of Listener
